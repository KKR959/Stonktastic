Module contents¶

The optimization part of the project that test a single stock for the optimum values to use in terms of cost/accuracy.

This intergrates heavily into the Jupyter notebook reports

  • optimizePolyReg : Code for optimizing Polynomial Regression

  • optimizeRanFor : Code for optimizing Random Forest
